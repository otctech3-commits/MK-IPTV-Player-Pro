# MK IPTV Player Pro

Free M3U/M3U8 IPTV player with audio switching, VLC mode, and multi-playlist support.

## Features
- ✅ Load multiple M3U playlists
- ✅ Add single M3U8 streams
- ✅ Audio track switching
- ✅ Player modes: Browser HLS, VLC-like, Open in VLC
- ✅ Search channels
- ✅ Watch history
- ✅ 100% Free, runs on GitHub Pages

## Deploy to GitHub Pages
1. Create repo `mk-iptv-player`
2. Upload all files
3. Settings → Pages → Deploy from `main` branch
4. Done! `https://username.github.io/mk-iptv-player`

## Usage
1. Click "Add Playlist" 
2. Paste M3U URL or upload file
3. Click any channel to play
4. Switch audio tracks if stream has multiple languages
5. Use "Open in VLC" for external player

## Legal
